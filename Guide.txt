1. Перейти по ссылке в браузере http://localhost/Tools/phpMyAdmin
2. Создать пустую базу данных с именем activerest1
3. Нажать на верхней панели кнопку импортировать и выбрать файл для импорта с расширением sql
4. Зайти в файл Connect.php и переиминовать строку $dbName = "activerest" в $dbName = "activerest1"
5. Пользоваться
